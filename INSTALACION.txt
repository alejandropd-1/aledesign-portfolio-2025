# MANUAL DE INSTALACIÓN Y CONFIGURACIÓN

## PASO 1: Instalar Node.js y npm
Si no tienes Node.js instalado, descárgalo desde: https://nodejs.org/

## PASO 2: Instalar las dependencias
Abre una terminal en la carpeta del proyecto y ejecuta:
```
npm install
```

## PASO 3: Copiar los estilos SASS
IMPORTANTE: Debes copiar todos los archivos SASS del proyecto original.

En Windows PowerShell, ejecuta:
```
xcopy /E /I C:\www\aledesign\sass\* C:\www\aledesign-portfolio-2025\src\styles\
```

O copia manualmente toda la carpeta 'sass' desde:
  C:\www\aledesign\sass
A:
  C:\www\aledesign-portfolio-2025\src\styles

Mantén la estructura de carpetas:
- abstracts/
- base/
- components/
- layout/
- utilities/
- vendor/

## PASO 4: Copiar las imágenes
Copia todos los archivos de imagen desde:
  C:\www\aledesign\public\images\
A:
  C:\www\aledesign-portfolio-2025\public\images\

Archivos necesarios:
- aledesign-avatar.png
- banner-background.png
- bg.png
- body-background.png
- grain.png

## PASO 5: Iniciar el servidor de desarrollo
```
npm run dev
```

El sitio estará disponible en: http://localhost:4321

## COMANDOS DISPONIBLES:
- npm run dev      # Inicia el servidor de desarrollo
- npm run build    # Construye el sitio para producción
- npm run preview  # Vista previa del sitio construido

## SOLUCIÓN DE PROBLEMAS:

Si encuentras errores con los estilos SASS:
1. Verifica que copiaste todos los archivos SASS
2. Verifica que la estructura de carpetas sea correcta
3. Reinicia el servidor de desarrollo

Si las imágenes no se muestran:
1. Verifica que copiaste todas las imágenes
2. Verifica que los nombres de archivo sean correctos
3. Revisa la consola del navegador para ver errores 404

## NOTAS IMPORTANTES:
- El proyecto usa Astro 4.16.18
- Los estilos están en SASS
- Los trabajos se agregan como archivos .md en src/content/jobs/
- El sitio es completamente estático y optimizado para SEO
